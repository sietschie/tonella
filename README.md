# Tonella
Tonella is a arduino based music box with a tangible user interface.

[tonella.goering.dev](https://tonella.goering.dev)

## Compilation
    pip install platformio
    cd src/tonella
    pio run
    pio run -t upload --upload-port /dev/ttyUSB1
